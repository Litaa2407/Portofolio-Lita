Project Detail

ORKESTRA

Platform manajemen keuangan organisasi mahasiswa dengan fitur pelaporan, OCR struk, dan integrasi backend penuh.

Ringkasan

ORKESTRA adalah aplikasi mobile untuk memudahkan organisasi mahasiswa mencatat pemasukan dan pengeluaran, memproses struk belanja secara otomatis, dan menghasilkan laporan keuangan yang jelas.

Peran Saya

  • Merancang UI mobile Flutter yang ramah pengguna untuk input transaksi dan laporan.
  • Membangun backend Spring Boot dengan endpoint autentikasi, data transaksi, dan laporan keuangan.
  • Menghubungkan aplikasi dengan PostgreSQL untuk penyimpanan data yang stabil.
  • Mengimplementasikan OCR menggunakan OpenRouter API untuk memindai struk secara cepat.

Highlight Fitur

• Input transaksi cepat

Menggunakan form yang bersih dan validasi untuk membantu pencatatan budget organisasi.

• Laporan otomatis

Generasi laporan bulanan yang mudah dibaca dan bisa dikirim ke ketua organisasi.

• OCR struk

Pemrosesan gambar struk secara otomatis untuk mempercepat input data tanpa ketik manual.

Teknologi

Flutter Spring Boot PostgreSQL OpenRouter API

Tantangan

Menyatukan mobile app dan backend dengan data real-time serta menjaga pengalaman pengguna tetap lancar saat memindai struk.

Hasil

Meningkatkan proses administrasi keuangan ormawa dan mempercepat laporan melalui sistem digital yang mudah dipakai.
