Project Detail

room.ukwms.ac.id

Sistem manajemen peminjaman ruang dan fasilitas kampus terpadu Universitas Katolik Widya Mandala Surabaya berbasis web.

Ringkasan

room.ukwms.ac.id adalah portal internal kampus yang mengotomatisasi peminjaman ruang kelas, auditorium, laboratorium, dan fasilitas penunjang di UKWMS. Proyek ini mendigitalisasi birokrasi peminjaman fisik yang sebelumnya rumit menjadi sistem reservasi mandiri (self-service) secara real-time untuk mahasiswa dan dosen.

Peran Saya (Full-Stack)

  • Frontend Next.js (SSR): Mengembangkan visualisasi jadwal okupansi ruangan interaktif berbasis Next.js dengan Server-Side Rendering (SSR) untuk memastikan render kalender ribuan jadwal perkuliahan berjalan instan dan SEO-friendly.
  • Backend API Golang: Merancang dan membangun RESTful API berkinerja tinggi menggunakan Golang (Gin framework) untuk menangani jutaan permintaan pemrosesan status reservasi ruangan secara real-time.
  • Pencegahan Konflik Peminjaman: Mengimplementasikan sistem locking dan penanganan konkurensi (Goroutines & Channels) di Golang untuk memvalidasi bentrok jadwal secara presisi pada tingkat database.
  • Integrasi Client-Server: Menghubungkan state management aplikasi Next.js dengan API Golang secara efisien serta menghubungkan backend ke database PostgreSQL.

Highlight Fitur

• Kalender Interaktif Next.js

Membantu civitas akademika melihat ketersediaan ruangan per jam dengan render antarmuka yang dinamis, responsif, dan ringan.

• Concurrency Room Locking

Perlindungan anti-double booking berbasis backend Golang untuk mengunci ruangan secara instan ketika ada dua pihak mengajukan di detik yang sama.

• Persetujuan Birokrasi Digital

Alur pengajuan izin secara otomatis diteruskan ke Kaprodi/Dekan terkait untuk persetujuan cepat.

Teknologi

Next.js Golang PostgreSQL Tailwind CSS

Tantangan

Menyajikan jadwal pemakaian ruang kuliah yang sangat padat secara akurat serta memitigasi bentrok peminjaman ruangan yang sering terjadi dalam pemrosesan bersamaan (concurrent booking).

Hasil

Mempersingkat alur birokrasi peminjaman dari rata-rata 2 hari menjadi kurang dari 5 menit secara digital.
